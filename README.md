# periodic-table-on-an-enum

This is a complete periodic table base on an enum containing all the elements and including the next fields:
* Atomic number
* Symbol
* Name
* Atomic mass
* CPK Color
* Electron configuration
* Electronegativity
* Atomic radius
* Ionization energy
* Electron affinity
* Oxidation states
* Standard state
* Melting point
* Boiling point
* Density
* Group block
* Year discovered
