# ptable

This is a complete periodic table for rust, including the following fields:
* Atomic number
* Symbol
* Name
* Atomic mass
* CPK Color
* Electron configuration
* Electronegativity
* Atomic radius
* Ionization energy
* Electron affinity
* Oxidation states
* Standard state
* Melting point
* Boiling point
* Density
* Group block
* Year discovered
